Bem-vindo(a) ao Simplescola

Tão simples quanto a página, assim é essa aplicação web;

Esse WEB App é bem simples, porém demonstra todas as funcionalidades simples de um CRUD de um texto qualquer, associado à uma chave para reconhecimento no banco. Aqui nós chamaremos essa chave de "Sala" e o texto em sí será a Cola.
A funcionalidade é bem simples: Basta informar o nome da sala no primeiro campo, e caso a sala você terá acesso ao conteúdo, assim como a opção de excluir a sala, caso deseje. Ao editar o texto, as alterações são salvas automaticamente no servidor.
É uma aplicação simples, porém demonstra a ação CRUD simples com um servidor feito com o framework NestJS (NodeJS), usufruindo dos serviços da Amazon.
